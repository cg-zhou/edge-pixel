【项目介绍】
Edge Pixel 是一个基于 ESA 边缘计算的高性能在线图像处理工具。

【创意卓越】
1. 采用现代化 UI 设计，视觉风格统一。
2. 提供直观的参数调整界面，支持多维度编辑。
3. 即时预览效果，满足快速处理的各种需求。

【应用价值】
1. 无需安装任何软件，打开即用。
2. Web应用易于分享，大幅降低用户学习成本和使用门槛。
3. 适用于社交媒体内容创作、电商产品图编辑等场景。

【技术探索】
1. 运用阿里云 ESA 边缘生态，通过 CDN 全球分发实现极速加载，展现 ESA 在实时应用中的优势。
2. 前端主要采用 Canvas API 与 CSS Filter 实现图像滤镜链式处理，代码精简高效。
3. 颗粒效果方面，采用2D伪随机数生成器实现，支持种子配置，逐像素进行图像处理。
4. 加载 Web 应用，仅需传输 494 KB（该大小包含示例图片）。

【功能特性】
1. 实时调整 - 支持多维度参数实时预览
2. 色彩调整 - 色相、饱和度、反色、灰阶
3. 亮度与对比 - 亮度、对比度、高光、阴影、光感
4. 效果处理 - 模糊、颗粒、褪色、暗角
5. 缩放 - 10% ~ 200% 范围内灵活缩放，实时显示像素尺寸
6. 旋转 - 支持 -180° ~ 180° 旋转范围，快速按钮支持 90° 旋转
7. 翻转 - 支持水平和垂直翻转
8. 多格式导出 - PNG（无损）、JPG（智能压缩）、WebP（现代高效压缩）
